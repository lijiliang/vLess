vLess

vLess是一个基于less的框架，其提供一些简洁常用的CSS写法

版本：v.0.1.0

.background-clip()
规定背景的绘制区域默认值为border-box

可用参数：

border-box 背景被裁剪到边框盒

padding-box 背景被裁剪到内边距框

content-box 背景被裁剪到内容框
.background-origin()
规定背景图片的定位原点即背景开始的位置。默认值为padding-box

可用参数：

border-box 从border区域开始显示背景

padding-box 从padding区域开始显示背景

content-box 从content区域开始显示背景
.background-size()
规定背景图像的尺寸默认值为auto

可用参数：

length 设置背景图像的高度和宽度。第一个值设置宽度，第二个值设置高度。如果只设置一个值，则第二个值会被设置为 "auto"。

percentage 用百分比来控制。第一个值设置宽度，第二个值设置高度。如果只设置一个值，则第二个值会被设置为 "auto"。

cover 把背景图像扩展至足够大，以使背景图像完全覆盖背景区域。(背景图像的某些部分也许无法显示在背景定位区域中。)

contain 把图像图像扩展至最大尺寸，以使其宽度和高度完全适应内容区域

transform 变形

.rotate(@degrees)

.rotateX(@degrees)

.rotateY(@degrees)

.rotateZ(@degrees)

.rotate3d(@x, @y, @z, @degrees)

rotate 旋转、@degrees参数默认值都为0deg依数传入值即可、通过 rotate() 方法，元素顺时针旋转给定的角度。允许负值，元素将逆时针旋转

应用实例

.rotate(60deg)定义 2D 旋转，在参数中规定角度

.rotateX(30deg)定义沿 X 轴的 3D 旋转

.rotatez(30deg)定义沿 Z 轴的 3D 旋转

.rotate3d(1, 2, 3, 10deg)定义 3D 旋转

.scale(@x, @y)

.scaleX(@x)

.scaleY(@y)

scale 缩放、参数默认值都为0通过 scale() 方法，以元素进行缩放，元素的尺寸会增加或减少，可根据给定的宽度（X 轴）和高度（Y 轴）参数传值

应用实例

.scale(2,.8)定义 2D 缩放转换，改变元素的宽度和高度

.scaleX(.8)定义 2D 缩放转换，改变元素的宽度

.scaleY(2)定义 2D 缩放转换，改变元素的高度

.translate(@x, @y)

.translateX(@x)

.translateY(@y)

.translateZ(@Z)

.translate3d(@x, @y, @z)

translate 移动、通过 translate() 方法，元素从其当前位置移动，根据给定的 left（x 坐标） 和 top（y 坐标） 位置参数

应用实例

.translate(10px,10px)定义 2D 转换，沿着 X 和 Y 轴移动元素

.translateX(20px)定义 2D 转换，沿着 X 轴移动元素

.translateY(20px)定义 2D 转换，沿着 Y 轴移动元素

.translateZ(20px)定义 3D 转化，仅使用用于 Z 轴的值

.translate3d(60px,20px,10px)定义 3D 转化,其参数对应X、Y、Z轴

.skew(@x, @y)

.skew(@x)

.skew(@y)

skew 倾斜、通过 skew() 方法，元素翻转给定的角度，根据给定的水平线（X 轴）和垂直线（Y 轴）参数

应用实例

.skew(10deg,10deg)定义 2D 倾斜转换，沿着 X 和 Y 轴

.skewX(10deg)定义 2D 倾斜转换，沿着 X 轴

.skewY(10deg)定义 2D 倾斜转换，沿着 Y 轴
transition 过渡

.transition(@property, @duration, @timing, @delay)

transition 过渡、过渡效果的简写属性，用于在一个属性中设置四个过渡属性

@property过渡的CSS属性名称

@duration过渡持续的时间

@timing过渡效果的时间曲线,时间曲线函数下面有介绍

@delay过渡效果的何时开始

例:.transition(background,1s,ease-in-out,6s)

.transitions(@string)

transition 多效果过渡、如需多个样式添加过渡效果，请添加多个属性，由逗号隔开，所以有了这个函数

例:.transitions(~'width .2s ease 3s,height .5s ease 6s')

.transition-property(@type)

过渡的 CSS 属性的名称默认值为all可传入需要过渡的属性：如width、height

.transition-duration(@type)

过渡效果持续的时间默认值为0s,可根据需要传值

.transition-timing(@transition-timing)

过渡时间函数默认值为ease

可传的参数有：linear、ease、ease-in、ease-out、ease-in-out、cubic- bezier(n,n,n,n)

.transition-delay(@type)

过渡延迟的时间默认值为0s,可根据需要传值
animations 动画

.animation(@animation)

animation 动画 其参数有：animation-name、animation-duration 、

animation-timing-function、 animation-delay 、 animation-iteration-count 、

animation-direction 、animation-play-state 所有的属性说明下面会一一讲解

使用：

单个动画效果：.animation(myfirst 5s linear 2s infinite alternate)

多个动画效果：.animation(~'myfirst 5s linear 2s infinite alternate,myfirst2 6s linear 2s infinite alternate')

.animation-name(@name)

动画的名称 是由Keyframes创建的动画名,必须传入动画名，否则会报错

.animation-duration(@duration)

播放动画所持续的时间长 可根据需要传值，如2s

.animation-timing-function(@timing-function)

动画时间函数

可传的参数有：linear、ease、ease-in、ease-out、ease-in-out、cubic- bezier(n,n,n,n)

.animation-delay(@delay)

元素动画开始时间需传入时间数，如2s

.animation-iteration-count(@iteration-count)

元素播放动画的循环次数可以取值为数字，其默认值为1、infinite为无限次数循环

.animation-direction(@direction)

元素动画播放的方向默认值为normal，如果设置为normal时，动画的每次循环都是向前播放；

另一个值是alternate，他的作用是，动画播放在第偶数次向前播放，第奇数次向反方向播放

.animation-play-start(@play-start)

控制元素动画的播放状态参数有：running和paused

runing 规定动画已暂停

paused 规定动画正在播放
